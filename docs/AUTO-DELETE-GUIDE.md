# 自动删除系统使用指南

## 系统概述
本系统自动删除顽固文件夹：`pre-architecture-backup-20260306-172408`

## 已创建的文件

### 核心文件
1. **`auto-delete-on-startup.ps1`** - 主删除脚本
   - 自动尝试多种删除方法
   - 记录详细日志到 `delete-log.txt`

2. **`setup-startup-delete.bat`** - 设置脚本
   - 创建启动项（登录后自动运行）
   - 清理旧脚本文件

3. **`run-auto-delete.bat`** - 一键运行脚本
   - 立即执行删除操作

## 使用方法

### 方案A：自动运行（推荐）
1. **运行设置脚本**
   ```cmd
   双击 setup-startup-delete.bat
   ```

2. **重启电脑**
   - 保存所有工作
   - 重启计算机

3. **自动执行**
   - 登录后系统会自动运行删除脚本
   - 启动项会自动删除自身

### 方案B：立即运行
1. **直接运行**
   ```cmd
   双击 run-auto-delete.bat
   ```

2. **或手动运行**
   ```cmd
   powershell -ExecutionPolicy Bypass -File auto-delete-on-startup.ps1
   ```

## 系统特点

### 智能删除
- 自动尝试多种删除方法
- 优先使用最有效的方法
- 失败后自动尝试替代方案

### 安全可靠
- 只删除指定文件夹
- 详细的操作日志
- 不会影响其他文件

### 自动清理
- 启动项自动删除自身
- 清理旧的脚本文件
- 保持工作空间整洁

## 预期结果

### 成功情况
1. 文件夹被完全删除
2. 日志文件记录成功信息
3. 磁盘空间进一步释放

### 如果失败
如果自动删除失败：
1. **查看日志文件** `delete-log.txt`
2. **尝试手动删除**（重启后立即执行）
3. **进入安全模式**删除

## 故障排除

### 常见问题
1. **脚本不运行**

2. **删除失败**

   - 进入安全模式删除
