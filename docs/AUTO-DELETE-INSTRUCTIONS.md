# 自动删除顽固文件夹系统

## 系统概述
本系统用于自动删除两个顽固的备份文件夹，这些文件夹由于路径过长或权限问题无法正常删除。

## 目标文件夹
1. `english-reorg-backup-20260306-234623`
2. `pre-architecture-backup-20260306-172408`

## 已创建的脚本文件

### 核心脚本
1. **`delete-after-reboot.ps1`** - 主删除脚本
   - 使用多种方法尝试删除
   - 包含robocopy、.NET、cmd等多种技术

2. **`setup-autodelete.bat`** - 设置脚本
   - 创建启动项（重启后自动运行）
   - 创建桌面快捷方式
   - 创建手动运行脚本

3. **`run-delete-now.bat`** - 立即运行脚本
   - 一键立即执行删除

## 使用方法

### 方案A：重启后自动删除（推荐）
1. 运行 `setup-autodelete.bat`
2. 重启电脑
3. 登录后系统会自动运行删除脚本

### 方案B：立即运行删除
1. 双击 `run-delete-now.bat`
2. 或双击桌面快捷方式"删除顽固文件夹"

### 方案C：手动运行
```cmd
cd C:\Users\luchaochao\.openclaw\workspace
powershell -ExecutionPolicy Bypass -File delete-after-reboot.ps1
```

## 删除技术说明

### 使用的删除方法
1. **robocopy镜像法** - 创建空目录镜像到目标文件夹
2. **.NET强力删除** - 使用System.IO.Directory.Delete
3. **cmd rd命令** - 传统的rd /s /q命令
4. **多重尝试** - 自动尝试多种方法直到成功

### 为什么这些方法有效
- **重启后运行**：避免文件被其他进程锁定
- **多种技术组合**：应对不同的删除障碍
- **自动重试**：一种方法失败自动尝试下一种

## 预期结果

## 安全说明
- 脚本只删除指定的两个备份文件夹
- 删除前会检查文件夹是否存在
- 所有操作都有详细日志
- 不会删除其他文件或文件夹
